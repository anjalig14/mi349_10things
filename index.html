Key Elements & Principles of Visual Design

Fundamental elements of visual design are basic tools that designers use in their work.
These elements include lines, shapes, negative space, volume, value, color, and texture.
These building blocks can be found in all products. In order to help us understand how
these elements should go together, we turn to design principles to guide us. These include
unity, gestalt, hierarchy, balance, contrast, scale, and dominance.
